#Basic Project

###Dependencies

* Gulp
* SCSS

### Installation

Clone the project in your workspace

	$ git clone git@github.com:JeremyFagis/BasicProject.git your-project
	$ cd your-project
	
Download packages

	$ (sudo) npm install
	
Compile assets

	#  compilation (watch & minification)
	$ gulp
	
	# Prod compilation (minification)
	$ gulp install
	
	# Dev compilation (watch & no-minification)
	$ gulp dev
